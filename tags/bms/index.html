Start my Blog!談談對網誌與接下來的規劃。

其實開網誌這件事大概在大二大三開始接觸github後就想試著弄弄看，但礙於課業、身心狀況以及自認內涵不足沒有太多可以分享的就一直被耽擱著，最近正因延畢以及一點生涯上的迷茫，覺得是時候可以把一些自己過去與未來所學的內容放在這裡分享，還有分享一些生活，希望這樣能夠讓自己比較不被社群軟體綁架，也希望帶給繁體中文創作者社群一點回饋。

我是誰

可以叫我Russell/陳柏昇，寫這篇網誌的當下待業中，正在重新參加駕訓班考取駕照中。從高中參加音樂遊戲社起接觸音樂遊戲約十年，一直有創作bms的衝動，所以廣泛涉獵了譜面製作、視覺設計(平面、插畫、3D動畫)、遊戲製作、數位音樂等技能樹。

網誌會拿來做甚麼

這個網誌主要會包含幾個大領域

1.音樂遊戲理論
類似於Ice的部落格內容，但會以自己的實力與經歷撰寫。

2.數位繪圖/動畫製作理論
現在其實有很多很多的網課可以學習(也很推薦參加，有其他人帶領通常更容易看到盲點)，所以會以自己的作品為例，談談製作的流程中有感到”Eureka”的瞬間。

3.數位音樂理論與製作
因為最近開始接觸FL Studio，發現英文的資源很多但中文意外的稀少，自己雖然走得很順但對一些中文非音樂班學生可能很不敢嘗試，所以也算是鋪一些路

4.生活雜談
可能像是這一篇一樣，隨意地談一些自己當時所想。

網誌會不會用AI生成?

我會使用AI，但在網誌中僅限於開發特定功能時使用。

對於網誌內容，我都會盡可能自己撰寫。因為我所看到願意使用AI創作的人，比起願意充實基礎理論後自行創作的人來的少太多了，我想鼓勵那些願意自己動筆、拿起樂器、開啟製譜器製作的人們。

你這樣不會擋到一些人的財路嗎?

大概會吧，我想內容有部分其實跟現在各種網課有所重疊。

但是比起擋財路現在待業中我也很窮不希望挨告，我更希望自己的文章能成為其他人教課時願意參考的對象，我只希望盡可能的傳遞正確的知識。因此，如果有我知道來源的引用內容我也會盡量加入。

結語

2026對我來說是各種層面上新的一年。希望今年能找到一些工作(各種領域都是)。

如果有緣看到這篇的人，希望以後的文章能夠在你創作學習的路上有所幫助。

lisyul skadi doula (It’s fun to learn about many things!)
